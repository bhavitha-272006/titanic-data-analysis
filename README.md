# Titanic Data Analysis Project

## Overview
Analysis of Titanic passenger data to find patterns in survival rate.

## Dataset
891 passengers with 12 columns including age, gender, class and fare.

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps
1. Data Loading
2. Data Cleaning — handled missing values in Age, Cabin, Embarked
3. Exploratory Data Analysis — 6 charts

## Key Findings
1. Only 38% of passengers survived
2. Females survived much more than males
3. 1st class passengers survived more than 3rd class
4. Higher fare = better survival chance
5. Age was not a strong factor in survival
